# SOC Portfolio

## Overview

This repository serves as a collection of cybersecurity investigations, threat analysis reports, detection engineering exercises, and hands-on security projects completed through practical learning, homelab environments, and security training platforms.

The objective of this portfolio is to demonstrate a structured approach to security monitoring, incident investigation, threat detection, and defensive analysis commonly performed within Security Operations Center (SOC) environments.

## Areas Covered

This portfolio includes investigations and practical exercises related to:

* Web Application Security

  * Cross-Site Scripting (XSS)
  * SQL Injection
  * Command Injection
  * Insecure Direct Object Reference (IDOR)
  * Local File Inclusion (LFI)
  * Remote File Inclusion (RFI)

* Security Operations & Incident Response

  * Alert Triage
  * Incident Investigation
  * IOC Analysis
  * Threat Hunting
  * Log Analysis

* Detection Engineering

  * SIEM Use Cases
  * Detection Rule Development
  * Alert Validation
  * MITRE ATT&CK Mapping

* Network & Threat Analysis

  * Traffic Analysis
  * Phishing Investigations
  * Malware Analysis
  * Threat Intelligence

## Methodology

Each case study is documented using a consistent investigation process:

1. Alert Review
2. Evidence Collection
3. Traffic and Log Analysis
4. IOC Identification
5. Threat Classification
6. MITRE ATT&CK Mapping
7. Incident Assessment
8. Findings and Recommendations

Where applicable, supporting screenshots, artifacts, detection logic, and investigation notes are included.

## Purpose

This repository is intended to showcase practical cybersecurity skills through documented investigations and hands-on defensive security exercises. The focus is on understanding attacker behavior, validating security alerts, and developing analytical workflows used in real-world SOC environments.

## Repository Status

This portfolio is actively maintained and will continue to grow with additional investigations, detection projects, threat analysis reports, and homelab-based security research.
